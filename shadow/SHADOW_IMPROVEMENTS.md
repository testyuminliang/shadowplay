| 部位 | 旧宽度 | 新宽度 |
|------|--------|--------|
| 手掌扇形 | 0.17 | 0.20 |
| 手掌横连 | 0.16 | 0.17–0.18 |
| 拇指 | 0.125–0.098 | 0.150–0.115 |
| 食/中指 | 0.108–0.090 | 0.130–0.105 |
| 无名指 | 0.102–0.084 | 0.122–0.098 |
| 小指 | 0.094–0.074 | 0.112–0.088 |

## 二、手势系统

### 2.1 九种动物手势定义

| 动物 | 手势规则 `[拇,食,中,环,小]` | 关键说明 |
|------|--------------------------|----------|
| 🐰 兔子 | `[F, T, T, F, F]` | 食指+中指竖直 |
| 🦊 狐狸 | `[T, T, F, F, T]` | 拇指+食指+小指张开 |
| 🐶 小狗 | `[T, T, F, F, F]` | 拇指+食指（枪形） |
| 🦆 鸭子 | `[T, T, T, T, T]` | 全展开，手侧对镜头 |
| 🐻 小熊 | `[T, T, T, T, T]` | 全展开，爪形微弯 |
| 🦢 天鹅 | `[F, F, T, T, T]` | 三指竖直，拇食指捏 |
| 🦋 蝴蝶 | `[T,T,T,T,T]` × 双手 | 需要双手，拇指交叉 |
| 🐌 蜗牛 | `[F, T, T, F, F]` | 食中指稍弯（非完全伸直） |
| 🐊 鳄鱼 | `[T,T,T,T,T]` × 双手 | 双手上下开合 |

> **注意**：鸭子/小熊/蝴蝶/鳄鱼的手指状态相同，视觉差异在于手的朝向和数量。用户选择对应动物后系统会给出不同的姿势提示。

### 2.2 手指伸展检测算法

```javascript
function detectFingers(lm) {
  // 拇指：指尖到手掌中心的距离 > 拇指IP到手掌中心距离 × 1.05
  const palmCx = lm[9];
  const thumbExt = dist(lm[4], palmCx) > dist(lm[3], palmCx) * 1.05;

  // 其余四指：指尖 y 坐标 < PIP y坐标 - 0.012（指尖明显高于近节关节）
  const indexExt  = lm[8].y  < lm[6].y  - 0.012;
  const middleExt = lm[12].y < lm[10].y - 0.012;
  const ringExt   = lm[16].y < lm[14].y - 0.012;
  const pinkyExt  = lm[20].y < lm[18].y - 0.012;
  return [thumbExt, indexExt, middleExt, ringExt, pinkyExt];
}
```

### 2.3 匹配打分

```javascript
score = (匹配手指数) / 5
// 阈值: score ≥ 0.88 （5 中至少 4.4 个匹配）
// 持续 ≥ 1.2 秒后确认成功，显示 "✓ 完美手势！"
```

### 2.4 双手手势

蝴蝶和鳄鱼检测双手：
- 如果 ≥ 2 只手被检测到，且两手合计伸展手指数 / 10 ≥ 0.88 → 成功
- 如果只检测到 1 只手，最高得分限制在 60%，提示用户需要双手

## 五、已知局限 & 后续优化方向

| 问题 | 原因 | 建议方案 |
|------|------|---------|
| 鸭子/熊/蝴蝶/鳄鱼指型相同 | 都是五指展开 | 加入手掌朝向检测（法向量） |
| 蜗牛 vs 兔子难区分 | 同样是食中指 | 检测手指弯曲角度（DIP vs PIP） |
| 拇指检测偶有误判 | 角度依赖手的旋转 | 改用与掌骨轴的夹角计算 |
| 蝴蝶需要双手对齐 | 无对齐检测 | 检测两手拇指是否接近 |
